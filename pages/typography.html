When to use
  • Large Title/Maps to iOS UINavigationBar large title and Android TopAppBar large title
  • Headline vs Body/Same 17px — Headline is 600 (emphasized), Body is 400 (default paragraph text)
  • Callout vs Subheadline/Same 16px — Subheadline is 500 (Medium) with +0.01em tracking, Callout is 400 (Regular)
  • Display sizes/Same values on all platforms — at ≥34px the pt/sp/px difference is negligible
Platform notes
  • All Bundle Inter on native — do not use Google Fonts runtime download. Android sizes match iOS 1:1 (sp = pt).
  • Web Small sizes bumped for non-Retina: subheadline 16px, footnote 14px, caption1 13px, caption2 12px (vs 15/13/12/11 native).
  • Web Small sizes get +0.01–0.02em letter-spacing. Always set explicit line-height — browser default ~1.2 is too tight.
