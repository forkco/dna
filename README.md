# FORK Design Language

Cross-platform design reference for iOS, Android, and Web.

Open `index.html` in a browser. No build tools, no dependencies beyond Google Fonts.

## Design DNA

Braun warmth · Apple clarity · Teenage Engineering boldness · Industrial tactility

## Key tokens

| Token | Light | Dark |
|-------|-------|------|
| `--label` | `#1A1A18` | `#F5F5F2` |
